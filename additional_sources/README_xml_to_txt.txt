1) Place xml_to_txt.py in the folder containing the directory of the .xml files.
2) Run without parameters
3) Specify the name of the directory containing .xml files when requested
4) The program outputs a .txt file (in the directory of xml_to_txt.py) with the same name of the directory specified in 3)


!!!NOTE!!!

When parsing the output .txt file it is better to check for inconsistencies between labels' names (e.g. 'hiddenboat' and 'hidden_boat')
